API 演示

通过 URL 参数直接生成二维码，例如：/api.html?text=Hello%20World&size=200

下载二维码

参数说明

  • text（必填）：要编码的内容。
  • size（可选）：图片宽高（像素），默认 200。
  • fg（可选）：前景色，CSS 颜色值，默认 #000。
  • bg（可选）：背景色，CSS 颜色值，默认 #fff。
  • ecc（可选）：容错级别，可选 L、M、Q、H，默认 M。
